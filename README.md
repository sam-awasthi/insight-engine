# Insight Engine

Turns customer complaints into marketing campaigns. Built at a hackathon — placed 3rd.

## What It Does

- Input a brand URL and category
- Aggregates complaints from Reddit, reviews, and social
- Extracts top pain points and objections
- Generates ad angles and short-form video scripts
- Exports to VEED for video production

## Framework: Listen → Script → Shoot

- **Listen:** Scrape and structure customer complaints
- **Script:** Turn pain points into ad angles and scripts  
- **Shoot:** Generate video content via VEED

## Tech Stack

- React + TypeScript
- Tailwind CSS
- Lovable (built at hackathon)
